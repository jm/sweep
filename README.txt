sweep
    by Jeremy McAnally
    http://github.com/jeremymcanally/sweep

== DESCRIPTION:

Rake, rebooted.

== FEATURES/PROBLEMS:

* Does everything Rake does except for rules and FileList (coming soon)

== SYNOPSIS:

Acts like Rake, only 1/10th of the code.

  namespace :thing do
    task :what do
      puts "wheeee"
    end
  end

== REQUIREMENTS:

* Ruby

== INSTALL:

* sudo gem install jeremymcanally-sweep
